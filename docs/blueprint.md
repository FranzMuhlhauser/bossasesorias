# **App Name**: BOSS Asesorías Web

## Core Features:

- Homepage Hero Section: Attractive opening section with key phrase, background image/video, and quick contact button.
- Service Showcase: Dedicated section to list and briefly describe BOSS Asesorías' core services, focusing on advisory and technical support rather than direct implementation.
- Company Overview: An 'About Us' section presenting the company's mission, vision, values, and team, designed with expandable sections for mobile-friendliness.
- Blog/Publications Section: A dynamic space for articles and news related to workplace wellness, occupational safety, and inclusivity.
- Contact Gateway: Integrated contact information with a clickable phone number, embedded map, and direct links to map applications.
- Floating WhatsApp Button: A persistent, floating button providing immediate access to BOSS Asesorías via WhatsApp.

## Style Guidelines:

- Primary color: Blue (#007BFF), mirroring the design system for trustworthiness.
- Background color: Light gray (#F8F9FA), taken from the design system, offering a clean, professional backdrop.
- Accent color: Cyan (#00D4FF) for highlights and interactive elements, drawing users' attention.
- Body and headline font: 'Montserrat' sans-serif, a clear, modern font used throughout the site, according to the design system. Note: currently only Google Fonts are supported.
- Maximum container width of 1200px to maintain readability.
- Consistent use of spacing, shadows, and border radii, as defined in the Design System.
- Subtle animations and transitions will enhance user engagement.
